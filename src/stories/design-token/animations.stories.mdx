# Les Durées d’Animations

<br />

## Animation

- Visez un délai de première interaction rapide de 300 ms ou moins avec le contenu ou les données primaires.
- Utilisez l'animation sur le premier CTA.
- Utilisez plus d'animations initiées par l'utilisateur que par le système.
- Utilisez notre palette de styles d'animation, de fonctions d'accélération et de variables de synchronisation

## When to use and when not to use

      </li>
      <li>

      </li>

        lorsque le rebond ou le dépassement est lié à l'effet d'animation ou au

    </ul>
  </div>

    <h4 class="text-red">when not to use</h4>
    <ul>

        non sur les contrôles d'interface.
      </li>
      <li>
        <b>Évitez d'empêcher les gens d'accéder à leur contenu</b> avec des

      </li>

          contenu

      </li>
      <li>
        <b>Évitez les animations trop rebondissantes</b> qui ternissent notre
        marque digne de confiance.
      </li>
    </ul>
